实验设计

沿用“设计 -> 审查完善 -> 通过落地 -> 严谨验收测试 -> 最终汇总报告”的循环：先验证冻结题库 SHA，再让学生侧基于普通 state-field SA、short_term_slot、residual B、lag successor、认知感受、情绪慢量和低粒度行动 trace 作答，最后由 Codex 按 judge_only rubric 事后评分。

本轮结论

Fresh300 teacher-off frozen-bank gate 通过：300/300，阈值 >=290/300。
结论边界：这证明的是 GL 冻结题库 teacher-off harness 在 Skill38 范围内达标；不证明 AP-Core runtime 持久记忆，也不证明广域开放世界中文对话成熟。
